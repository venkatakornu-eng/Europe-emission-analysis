# Europe-emission-analysis
This project analyzes greenhouse gas emissions across European countries using R. A t-test is applied to compare mean emission values between selected countries to identify statistically significant differences. The analysis supports understanding emission patterns and sustainability trends.

# 🌍 Europe Emission Analysis

# 📌 Project Overview

  This project analyzes greenhouse gas emissions across European countries using data from Eurostat.
  The goal is to explore emission trends, compare countries, and derive insights related to sustainability and climate transition.

# 📊 Dataset Description

Source: Eurostat (env_ac_ainah_r2)

Original dataset contained 65 sheets covering multiple gases and sectors

Selected 20 relevant sheets focused on CO₂ and greenhouse gas emissions

Consolidated into a clean and structured Excel dataset for easier analysis and visualization

# 🛠️ Tools & Technologies

R (Data Analysis, Statistical Modelling)

Excel (Data Cleaning & Preprocessing)

Eurostat Dataset

# ⚙️ Workflow
1. Data Cleaning & Preparation

Removed missing and inconsistent values

Converted data formats (TSV → Excel)

Structured dataset for analysis

2. Data Transformation

Merged selected sheets into a single dataset

Normalized values for comparison across countries

3. Exploratory Data Analysis (EDA)

Trend analysis of emissions over years

Country-wise comparison 

Distribution analysis using boxplots

4. Statistical Analysis

Correlation analysis (Year vs Emission)

Linear regression modelling

Hypothesis testing (t-test)

# 📈 Key Insights

Emission trends vary significantly across European countries

Some countries show gradual reduction, while others fluctuate

Time alone is not a strong predictor of emissions → multiple external factors influence trends
